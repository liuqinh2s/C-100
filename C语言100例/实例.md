### 实例20

#### 题目

猴子吃桃问题：猴子第一天摘下若干个桃子，当即吃了一半，还不瘾，又多吃了一个 第二天早上又将剩下的桃子吃掉一半，又多吃了一个。以后每天早上都吃了前一天剩下 的一半零一个。到第10天早上想再吃时，见只剩下一个桃子了。求第一天共摘了多少。

#### 代码

```c
#include <stdio.h>

int main(void){
    float sum = 0;
    float h = 100;
    sum += h;
    for(int i=1;i<10;i++){
        h = h/2;
        sum += 2*h;
    }
    printf("共经过%f米，第10次反弹%f米\n", sum,h/2);
    return 0;
}
```

#### 答案

```
共经过299.609375米，第10次反弹0.097656米
```
