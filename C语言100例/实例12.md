### 实例12

#### 题目

判断101到200之间的素数。

#### 代码

```C
#include <stdio.h>
#include <math.h>

int main(void){
    for(int i=101;i<200;i++){
        for(int j=2;j<=(int)sqrt(i)+1;j++){
            if(j==(int)sqrt(i)+1){
                printf("%d\n",i);
            }
            if(i%j==0){
                break;
            }
        }
    }
    return 0;
}
```
