### 实例12

#### 题目

判断101到200之间的素数。

#### 代码

```C
#include <stdio.h>
#include <math.h>

int main(void){
    printf("请输入一个整数：\n");
    int n;
    scanf("%d", &n);
    int i;
    for(i=2;i<(int)sqrt(n);i++){
        if(n%i==0){
            printf("不是素数\n");
            break;
        }
    }
    if(i==(int)sqrt(n)){
        printf("是素数\n");
    }
    return 0;
}
```
