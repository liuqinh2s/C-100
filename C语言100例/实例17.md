### 实例17

#### 题目

输入一行字符，分别统计出其中英文字母、空格、数字和其它字符的个数。

#### 代码

```c
#include <stdio.h>

int main(void) {
    int letters = 0;
    int spaces = 0;
    int numbers = 0;
    int others = 0;
    char c;
    printf("请输入一行字符\n");
    while((c=getchar())!='\n'){
        if(('a'<=c && c<='z') || ('A'<=c && c <= 'Z')){
            letters++;
        }else if(c == ' '){
            spaces++;
        }else if(c<='9' && c>='0'){
            numbers++;
        }else{
            others++;
        }
    }
    printf("字符串中英文字母%d个，空格%d个，数字%d个，其他字符%d个\n", letters,spaces,numbers,others);
    return 0;
}
```
