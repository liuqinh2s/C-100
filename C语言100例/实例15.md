### 实例15

#### 题目

利用条件运算符的嵌套来完成此题：学习成绩>=90分的同学用A表示，60-89分之间的用B表示，60分以下的用C表示。

#### 代码

```c
#include <stdio.h>

int main(void){
    printf("请输入成绩：\n");
    int n=0;
    scanf("%d", &n);
    if(n>=90){
        printf("A\n");
    }else if(60<=n && n<=89){
        printf("B\n");
    }else if(n<60){
        printf("C\n");
    }
    return 0;
}
```
